عمل sorting لل array بلغة python

لنفترض أن لدينا ال list الآتيه ونريد تطبيق الطرق عليها

numbers = [3, 9, 1, 0, 2, 5, 8]

الطريقة الأولى

استخدام ال ()reverse

      numbers.reverse()

النتيجة: [9, 8, 5, 3, 2, 1, 0]

لاحظ أن reverse لا تقوم بإرجاع قيمة (array جديدة) فإذا قمنا بتجربة الكود الآتي فستكون التنيجة None

      new_list = numbers.reverse()
      print(new_list)

الطريقة الثانية

استخدام for

      for n in range(len(numbers) // 2):
        numbers[n], numbers[-1 - n] = numbers[-1 - n], numbers[n]

في هذا المثال قد قمنا بإيجاد نقطة المنتصف لل array ثم بعد ذلك استبدال كل عنصر من عناصر النصف الأول بما يقابله من عناصر النصف الثاني

النتيجة: [9, 8, 5, 3, 2, 1, 0]

الطريقة الثالثة

استخدام for

هنا طريقة أخرى لعمل sorting لل array باستخدام for

      for n in range(len(numbers)):
        last_n = numbers.pop()
        numbers.insert(n, last_n)

في هذا المثال قمنا بعمل loop على ال array، وفي كل مرة نقوم باستخراج آخر عنصر في ال array وتخزينه في variable، ومن ثم استخدام insert التي تأخذ منك رقم ال index الذي تريد وضع عنصر معين بعده، فنقوم بمرير ال index الخاص بكل عنصر في array ومن ثم وضع العنصر الذي استخرجناه في بداية ال array

النتيجة: [9, 8, 5, 3, 2, 1, 0]

الطريقة الرابعة

استخدام reversed

      list(reversed(numbers)

reverse تقوم بعكس ترتيب العناصر داخل ال array ، ولكنها لا تقوم بإرجاع قيمة ، أي أنه لا يمكنك تخزينها في variable كما بالمثال الآتي : 

      new_list = list(reversed(numbers))

بعد ما نقوم بعمل reverse لل array نقوم بتحويل النتيجة للنوع list ، لأن reversed لا تقوم بإرجاع بيانات نوعها list

النتيجة: [9, 8, 5, 3, 2, 1, 0]

الطريقة الخامسة

استخدام ال slicing

      new_list = numbers[::-1]

يمكننا استخدام ال [first_index:last_index:steps] لعمل reverse لل array

في المثال السابق لم نكتب ال index البدائي ولا النهائي ، وهذا معناه أننا سنأخذ جميع العناصر الموجودة في ال array، الرقم -1 في ال array يشير أننا سنمر على العناصر عكسيا ( البداية من النهاية والنهاية من البداية )
